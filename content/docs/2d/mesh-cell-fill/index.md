## Inputs

- Mesh cell (Mc) - [geometry] - Lattice cell unit to map to mesh faces
- Mesh faces - (Mf) - [mesh] - Mesh face (exploded as single meshes)

## Outputs

- Lattice Hinge (L) - [curve] - Joined curves that make up the lattice hinge
