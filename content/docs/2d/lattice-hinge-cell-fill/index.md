## Inputs

- Unit cell (U(c)) - [curves] - Unit cell to populate lattice hinge panels
- Panels (P) - [surface] - Panels to be populated with unit cells

## Outputs

- Lattice Hinge (L) - [curve] - List of curves that make up the lattice hinge
